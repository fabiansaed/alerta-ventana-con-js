Comandos Linux para actualizar proyecto:

git add .
*Preparar todos los archivos para enviar

git commit -m "Mi mensaje"
*Guardará en el repositorio local los cambios realizados y se incluye un mensaje

git push -u origin main
*Cargar el contenido del repositorio local al repositorio remoto en github
